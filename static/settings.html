系统设置

配置保存在 MySQL 表 app_runtime_settings（与 MYSQL_DATABASE 同库，连接参数仅读 .env 中的 MYSQL_*），优先级高于 .env 中的业务项（未入库项仍读环境变量）。需同时配置 MYSQL_HOST 与 AIPRINTWORD_ADMIN_TOKEN 后方可保存。若通过局域网 IP 访问，口令必须在运行 Flask 的那台电脑的 .env 或 data/admin_token.txt 中配置，仅在你自己电脑的 .env 里改是无效的。
